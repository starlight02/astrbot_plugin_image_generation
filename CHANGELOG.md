  - 兼容旧版显示信息开关，自动迁移为新的结果信息选项。

- **v1.1.7-2026-05-25**
  - 升级提示词和图片 AI 审核默认提示词，改用 XML 分区结构并放宽默认阻止范围。
  - 增强审核模型输出解析，支持 Markdown JSON 代码块、XML 标签包裹和混合文本中的 JSON 结果。
  - 重构配置管理模块，拆分默认配置和迁移逻辑，并兼容旧版内置审核提示词自动迁移。

- **v1.1.6-2026-05-25**
  - 将默认宽高比和默认分辨率改为“不指定”，请求中不再携带对应字段。
  - 新增默认分辨率“不指定”选项，并兼容旧配置中的“自动”。
  - 优化各图像适配器的可选参数构造逻辑，避免向上游发送未指定的比例或分辨率字段。

- **v1.1.5-2026-05-24**
  - 调整 LLM 生图工具参考图收集策略：默认不再自动读取当前消息图片、引用图片和 @ 用户头像，避免与显式参考图或人设参考图重复传入。
  - LLM 生图工具在提交生图任务前自动去除重复参考图。
  - 优化@获取头像逻辑

- **v1.1.4-2026-05-23**
  - 修复 LLM 生图工具处理显式参考图时的日志前缀变量遮蔽问题，避免 `reference_images` 触发 `'str' object is not callable`。

- **v1.1.3-2026-05-22**
  - 新增使用限制豁免配置：支持 AstrBot 管理员无视使用限制，并可配置额外的使用限制白名单 UMO。
  - 管理员和白名单会话可跳过会话黑名单、频率限制和每日额度限制，但仍保留每日用量记录。
  - 优化每日用量展示：豁免会话在启用每日额度时显示为 `n/∞`。

- **v1.1.2-2026-05-21**
  - 重构配置解析器：支持旧字段自动迁移、按配置模板规范化配置、移除废弃项并补齐缺失项。
  - 优化配置模板描述文本，统一“供应商”等术语，精简过时说明并提升配置项可读性。
  - 重构日志系统：新增统一日志工具与模块前缀，规范 task_id 链路日志，集中处理敏感信息脱敏与日志摘要。
  - 加强日志隐私保护：避免输出完整提示词、用户标识、URL、本地路径、data URL、第三方响应体和 provider 错误体。
  - 降低后台任务日志噪音，统一异常日志风格，并补充主要生成、审核、下载和适配器调用路径的安全日志处理。

- **v1.1.1-2026-05-21**
  - 将 `z_image_gitee` 从 Z-Image 专用适配器改为 `gitee_ai` Gitee AI 通用图像接口适配器，支持 `/v1/images/generations` 与 `/v1/images/edits`，并保留旧配置键兼容。
  - Gitee AI 适配器新增图生图接口选择：默认自动按模型选择 generations/edits，也可手动固定指定接口；配置模板仅保留必要连接、模型和能力选项。
  - 更新 Gitee AI 默认模型列表，并按 Gitee AI edit/generation 模型清单优化自动接口识别。
  - 新增配置加载迁移检查，自动将旧版配置键转换为新版格式、补齐默认项并清理弃用字段。

- **v1.1.0-2026-05-21**
  - 新增人设模板：可在配置中维护人设名称、固定提示词和可选参考图，并通过 `/生图 <人设名称> [额外提示词]` 或 LLM 生图工具直接使用。
  - 将提示词模板配置统一到 `prompt_templates`，预设迁移为 `prompt_templates.presets`，人设模板放入 `prompt_templates.personas`。
  - 重构 LLM 工具：保留统一的 `generate_image`，覆盖普通生图、预设生图、人设/自拍生图和图生图，并支持 `preset`、`persona`、`reference_images`、`avatar_references` 等参数。
  - 新增独立的 `query_image_presets` 和 `edit_image_presets` 工具，分别用于查询预设/人设列表与详情，以及创建、删除预设。
  - LLM 工具开关改为可多选的工具列表，可分别启用生图、预设查询和预设编辑能力。
  - 优化参考图收集：可同时使用人设参考图、显式参考图、消息图片、引用消息图片、@ 用户头像和指定用户头像。
  - 完善本地图片来源解析：支持 Linux/Windows 绝对路径、`file://` 文件 URL、HTTP(S) 图片 URL，并兼容 AstrBot 文件配置返回的插件数据 `files/...` 路径。

- **v1.0.7-2026-05-19**
  - 新增火山方舟适配器 `volcengine_ark`，支持 Seedream 图片生成接口 `/api/v3/images/generations`。
  - 支持火山方舟文生图、图生图、组图参数、自动尺寸映射和可选联网搜索配置。

- **v1.0.6-2026-05-18**

  - 新增 SiliconFlow 配置模板，内置 Kolors、Qwen-Image、Qwen-Image-Edit 系列模型，按模型自动处理推荐尺寸与图片下载。

  - 新增开始生图任务提示模板配置，支持自定义提示文本及参考图数量、提示词、预设、任务 ID、模型等占位符。
